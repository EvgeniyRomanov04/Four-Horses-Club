# Адаптивный лендинг — Тестовое задание

Вёрстка адаптивного лендинга по макету Figma с использованием чистого HTML, CSS и JavaScript.  
Без сторонних библиотек и фреймворков.

## 📋 Требования задания

- ✅ **Pixel Perfect** — точное соответствие макету
- ✅ **Адаптивность** — корректное отображение на любых экранах (нет горизонтального скролла, текстовые элементы не наезжают друг на друга)
- ✅ **Нет дублирования текста** — разные версии используют уникальный контент
- ✅ **Бегущая строка** — работает корректно
- ✅ **Якорные кнопки** — на стартовом экране ведут к соответствующим блокам
